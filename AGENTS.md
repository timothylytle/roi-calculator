# Project Overview
This is a [Next.js](https://nextjs.org) web app project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

This project offers a couple of ROI (return on investment) calculator for a SaaS server. The happ is hosted in Vercel as a serverless web app.
